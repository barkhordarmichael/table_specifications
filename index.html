HTML Table reference

HTML Attributes

Tag Name Description
<img> img Insert image between tags <img src="image.jpg">
<address> address Defines contact information for the author, requires a closing tag
<a> anchor To link to a file or website.<a href="url or internal link"><a>
<ul> Unordered list Creates an unordered list with nested <li> tag
<input> Input Insert input field by specifing input type
<form> Form field Inserts form field to collect user input
